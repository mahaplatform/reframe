Quickstart

First, add reframe to your project

{% highlight bash %}
npm install --save reframe
{% endhighlight %}

Next create your first component

{% highlight javascript %}
import { Collection } from 'reframe';

export default class Contacts extends React.Component {

  render() {
    return 
  }

  _getCollection() {
    return {
      id: 'people',
      records: [
        { name: 'Greg Kops' },
        { name: 'Megan Pugh' },
        { name: 'Kath Tibbetts' },
        { name: 'Armand Zerilli' }
      ]
    }
  }

}
{% endhighlight %}

Compile your application and hit your browser:

It's that easy! Of course, this isn't a very interesting example as it merely displays a simple grid of static data. What if we wanted to load the data from a remote data source? Ignoring security and authentication concerns for a moment, let's refactor our collection:

{% highlight javascript %}
import { Collection } from 'reframe';

export default class Contacts extends React.Component {

  render() {
    return 
  }

  _getCollection() {
    return {
      id: 'people',
      records: 'https://api.example.com/contacts'
    }
  }

}
{% endhighlight %}

Not much will change with the interface, but we made an important change to the component's configuration. By replacing our static json records object with an API endpoint: Now, the component will attempt to load this data remotely!

Let's take a look at at forms:

{% highlight javascript %}
import { Form } from 'reframe';

export default class New extends React.Component {

  render() {
    return 

  }

  _getForm() {
    return {
      id: 'new-contact',
      sections: [
        {
          fields: [
            { code: 'name', label: 'Name', type: 'textfield' },
          ]
        }
      ]
    }
  }

}
{% endhighlight %}

Compile your application and hit your browser:

Once again not a lot to look at yet in this example, and if you press the save button, nothing will happen. Just take a moment to notice how declarative Reframe configuration is. Just as with the collection, you can describe the look, functionality, and behavior all in one JSON object. As you might imagine, connecting this form to a remote datasource is a trivial addition:

{% highlight javascript %}
import { Form } from 'reframe';

export default class New extends React.Component {

  render() {
    return 
  }

  _getForm() {
    return {
      id: 'new-contact',
      method: 'POST',
      action: 'https://api.example.com/contacts',
      sections: [
        {
          fields: [
            { code: 'name', label: 'Name', type: 'textfield' },
          ]
        }
      ]
    }
  }

}
{% endhighlight %}

Now your form is able to submit an ajax request to your server API with two simple additions to your configuration. Of course, you'll need to handle validation on both the client side and server side, but we've thought through all that! Reframe components are intended to make integrating with your server backend a snap!
