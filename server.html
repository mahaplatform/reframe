Server Requirements

Collection Response

When using the Collection or Container components to retrieve multiple resources from an endpoint, Reframe expects the records to be returned using the following format:

{% highlight javascript %}
{
  records: [
    { first_name: 'Greg', last_name: 'Kops' },
    { first_name: 'Megan', last_name: 'Pugh' },
    { first_name: 'Kath', last_name: 'Tibbetts' },
    { first_name: 'Armand', last_name: 'Zerilli' }
  ],
  total_records: 225,
  total_pages: 5,
  current_page: 1
}
{% endhighlight %}

Resource Response

When using the Container component to load a singleton resource, Reframe expects you to return the resource's attributes in a flat json object:

{% highlight javascript %}
{
  first_name: 'Greg',
  last_name: 'Kops'
}
{% endhighlight %}

Error Response

When using the Form component to create or update records on the server, Reframe expects your endpoint to return errors in the following format:

{% highlight javascript %}
{
  errors: {
    first_name: [
      "field is required"
    ],
    last_name: [
      "field is required"
    ],
  },
  message: "There was a probem with your input"
}
{% endhighlight %}
