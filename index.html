Getting Started with Reframe

What is Reframe?

Reframe is a collection of components which help developers rapidly prototype and build responsive web applications. Built upon open source libraries like React, Redux, and Semantic UI, Reframe is meant to coexist and integrate within the existing React ecosystem

How do I use Reframe?

A simple collection

{% highlight javascript %}
import { Collection } from 'reframe';

class Index extends React.Component {

  render() {
    return 
  }

  _getCollection() {
    return {
      id: 'people',
      columns: [
        { label: 'First Name', key: 'first_name', visible: true },
        { label: 'Last Name', key: 'last_name', visible: true }
      ],
      records: [
        { first_name: 'Greg', last_name: 'Kops' },
        { first_name: 'Armand', last_name: 'Zerilli' },
        { first_name: 'Megan', last_name: 'Pugh' },
        { first_name: 'Kath', last_name: 'Tibbetts' }
      ]
    }
  }

}

export default Index
{% endhighlight %}

A simple form

{% highlight javascript %}
import { Form } from 'reframe';

class Index extends React.Component {

  render() {
    return 

  }

  _getForm() {
    return {
      id: 'contact-new',
      sections: [
        {
          label: 'Basic Information',
          fields: [
            {
              code: 'first_name',
              label: 'First Name',
              type: 'textfield',
              required: true
            },
            {
              code: 'last_name',
              label: 'Last Name',
              type: 'textfield',
              required: true
            }
          ]
        }
      ]
    }
  }

}

export default Index
{% endhighlight %}
